# Exploratory Analysis of Articles Using NotebookLM

Perform exploratory analyses on scientific articles and related documents by using **NotebookLM**. 
This exercise will help you organize information, identify key concepts, and visualize relationships between ideas.|

Time: 10 min
Grouping: individual activity

## Step 1: Collect Source Material
Perform a search using a tool of your choice (e.g., AI tool of choice, or Google Scholar, PubMed) to find **three articles or documents** relevant to your research topic.  
Requirements:
- Files must have an **open license** (e.g., CC-BY).
- If you cannot find suitable files, use the provided example documents.

## Step 2: Upload to NotebookLM
- Create a **new notebook** in NotebookLM.
- Upload the three selected files.
- Ensure the documents are properly indexed so NotebookLM can process them.

## Step 3: Generate a Mind Map
- Use NotebookLM’s summarization and visualization features to create a **mind map** that:
  - Highlights the main concepts from the documents.
- Take a moment to reflect on how this visualization supports exploratory analysis.
