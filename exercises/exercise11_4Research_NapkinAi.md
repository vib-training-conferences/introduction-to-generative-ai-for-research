# Exercise 11:

Create an illustration with Napkin.AI (free version)

- Open your browser and surf to https://app.napkin.ai 

1. Create a blank napkin
2. Enter a title e.g. Curriculum topics 
3. Write a prompt or enter the list below:

```
   dataset representation
   computational dataset creation
   dataset collection / FAIRification
   evaluation method creation
   data augmentation (synthetic)
   model development
```
 
4. Download a PDF

5. Import in Inkscape to get access to the individual elements to refine/redraw
